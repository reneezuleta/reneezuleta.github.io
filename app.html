Perspectives:

The app can be utilized in several ways by selectively enabling the inclusion of a metronome, percussion tracks and melody tracks. Here are some perspectives on the app usage:

  • Beat Practice: practice drumming techniques by playing along preloaded drum grooves. If you need additional groove options, you can use the editor to create and save additional patterns.
  • Drum play along: practice drumming techniques by playing along melody tracks.
  • Band Style: play along the drum tracks and some or all the Melody tracks. Four melody tracks are available. They are all programmable to choose an instrument, velocity and middle octave. Plus, they can be silenced if desired.
    • Track 1 which plays a computer generated set of notes.
    • Track 2 which also plays a computer generated set of notes (following or diverging Track 1)
    • Track 3 which plays a chords pattern for the selected chord progression (randomly selected) .
    • Track 4 which is designed to play along the percussion bass track.
  • MIDI Generator: track sequences can be saved to your device or uploaded to the cloud for further use.
    • So that you can reload them at a different time.
    • So that you can import them on your favorite music editor for further processing and conversion to other audio formats.
  • Song Composer:
    • A set of patterns can be arranged together to create a song.
    • Songs can be saved and reloaded for later reuse.
    • songs can be created in two ways:
      • 1. Manual: When you hear a pattern you like you can add it to your song buy activating the +Song button.
        • You can add as many patterns as you like, you will be able to rearrange them and edit them.
      • 2. Automatic: When you hear a pattern you like, enter Song Mode, then one ins the Song Mode view activate the new button
        • A template of many song parts will be generated, again you will also be able to rearrange them and edit them.

Home Page:

The Home page includes the basic interface to start a new play.

  • The navigation bar on top contains:
    • Home button, to return to top of view.
    • Gear Icon that will take you to the Settings Screen.
    • Song Mode toggle.
    • Question Mark Icon that can be used to toggle Help Mode.
      In this mode you can tap the different items on the screen and read their purpose.
    • +Song button to add the current play to the song components<\li>
  • Right below the navigation bar find a row of buttons including:
    • Restart play.
    • Play/Pause.
    • MIDI file download: you can download MIDI files and play them.
    • Create and upload a batch of 10 MIDI files which are randomly created at request time.
    • Upload the current MIDI sequence.
  • The next row contains the Mode selector
    When you change the mode a new sequence is created with the current settings.
    • Preset Mode: the MIDI sequence is constructed using the groove, drum and melody settings.
      In the Home screen, select Preset, then tap the ? icon in the navigation bar to toggle Help Mode.
      Tap the items to read the help description for all the options.
    • Lotto Mode: the MIDI sequence is constructed using randomly selected groove parameters and the user selected melody parameters.
      In the Home screen select Lotto, then tap the ? icon in the navigation bar to toggle Help Mode.
      Tap the items to read the help description for all the options.
  • The play progress bar shows the play status. You can advance or rewind they play sequence by adjusting the slider.
  • Below the progress bar, there are several action buttons to randomly select new play parameters.
  • The description text shows the play sequence parameters.
  • The play control switches can select which tracks to include as well as if we want to create a play that offers some variation.
    (Again, look at Help Screen on the device)
  • The loop counter just selects how many times to repeat the groove sequence.
    Note that if with variations is selected and not in Song mode, this value will be controlled by the sequencer, and you cannot make a selection.
    Once the music player is loaded the music play time is shown.
  • The tempo slider controls the beats per minute of the current play sequence.
  • The Tab bar buttons at the bottom take you to the different setting sections.
    • The Groove, Drums and Melody buttons take you the particular settings section
    • The Editor button takes you to the groove editor
    • The Song button takes you to the Song editor
    • The Sound button takes you the the Percussion and Melody Profile Selectors

Perspectives:

The app can be utilized in several ways by selectively enabling the inclusion of a metronome, percussion tracks and melody tracks. Here are some perspectives on the app usage:

  • Solo Drummer: practice drumming techniques by playing along preloaded drum rhythms. If you need additional rhythm options, you can use the editor to create and save additional patterns.
  • Drummer Plus: practice drumming techniques by playing along melody tracks that fit within the selected rhythm.
  • Band Style: play along the drum tracks and some or all the Melody tracks. Four melody tracks are available. They are all programmable to choose an instrument, velocity and middle octave. Plus, they can be silenced if desired.
    • Track 1 which plays a computer generated set of notes.
    • Track 2 which also plays a computer generated set of notes (following or diverging Track 1)
    • Track 3 which plays the chords for the selected chord progression.
    • Track 4 which is designed to play along the percussion Base track.
    • Backing Generator: track sequences can be saved to your device or uploaded to your cloud for further use.
      • So that you can reload them at a different time.
      • So that you can import them on your favorite music editor for further processing and conversion to other audio formats.

Home Page:

The Home page includes all the basic interface to start a new play.

  • The navigation bar on top contains:
    • Title showing the operating mode.
    • Question Mark Icon that can be used to toggle Help Mode.
      In this mode you can tap the different items on the screen and read their purpose.
    • Gear Icon that will take you to the Settings Screen.
  • Right below the title are the action buttons:
    • Restart play.
    • Play/Pause.
    • MIDI file upload: you can download MIDI files and play them.
    • Create and upload a batch of 10 MIDI files which are randomly created at request time.
    • Upload the current MIDI sequence.
  • Below the action buttons there are three Mode selector buttons
    When you tap any this buttons a new sequence will be created with the current rhythm parameters.
    Note that melody parameter changes are always incorporated into the play sequence, there is no need to tap the mode.
    Since rhythm determines the sequence timing we ask the user to confirm the change by tapping the mode button.
    • Preset Mode: the MIDI sequence is constructed using the rhythm, drum and melody settings.
      In the Home screen, select Preset, then tap the ? icon in the navigation bar to toggle Help Mode.
      Tap each Rhythm, Drums and Melody selectors in the bottom tab to read the help description for all the options.
    • Random Mode: the MIDI sequence is constructed using randomly selected rhythm parameters and the user selected melody parameters.
      In the Home screen select Random, then tap the ? icon in the navigation bar to toggle Help Mode.
      Tap each Rhythm, Drums and Melody selectors in the bottom tab to read the help description for all the options.
    • Arrangement Mode: the MIDI sequence follows the selected arrangement. An arrangement is created by saving a list of rhythms and one set of melody parameters to apply to all the rhythms in the arrangement list.
      The arrangement also specifies the tempo to use.
      When you select Arrangement mode, you can tap the new rhythm button to create a random arrangement set:
  • The play progress bar shows the play status. You can advance or rewind they play sequence by adjusting the slider.
  • The description text shows the elements of the play sequence.
  • The play control switches can select which tracks to include as well as if we want to create a play that offers some variation.
    (Again, look at Help Screen on the device)
  • The loop counter just selects how many times to repeat the rhythm sequence.
    Note that if with variations is selected and not an arrangement this value will be controlled by the sequencer, and you cannot make a selection.
    Once the music player is loaded the music play time is shown.
  • The tempo slider controls the beats per minute of the current play sequence.
  • The Tab bar buttons at the bottom can take you to the different setting pages, to the arrangement editor and to the sound profile selector.
    • The Rhythm, Drums and Melody take you the particular available settings for the mode in use (Preset, Random or Arrangement)
    • Arrange takes you to the Arrangement Selector / Editor
    • Sounds takes you the the Sound Profile Selectors
